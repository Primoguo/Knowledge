- 所有可替换的外部能力（TTS、AI 摘要、语音克隆等）均以 protocol 定义边界（如 SpeechSynthesizerProtocol），具体实现类遵循该协议，上层仅依赖协议而非具体类型。
- 跨进程/跨生命周期共享的状态（ErrorHandler、ShareExtensionHandler、ThemeManager、AudioSessionService 等）统一以 static let shared = ... 单例形式暴露，并在需要时通过 @StateObject/@ObservedObject 注入到 SwiftUI 视图。
- SwiftData 模型使用 @Model 标注，原始存储字段以 fileTypeRaw 等命名，再通过同名计算属性转换为强类型枚举（如 DocumentType），保持模型与显示逻辑分离。
- 异步错误统一经 ErrorHandler.shared.currentAlert 发布 Alert，视图侧通过 .alert(item:) 集中消费，避免在各处分散处理 try/catch。
- UI 更新一律通过 DispatchQueue.main.async 或 await MainActor.run 切回主线程，确保 AVSpeechSynthesizerDelegate 回调与网络回调不会在主线程阻塞。
