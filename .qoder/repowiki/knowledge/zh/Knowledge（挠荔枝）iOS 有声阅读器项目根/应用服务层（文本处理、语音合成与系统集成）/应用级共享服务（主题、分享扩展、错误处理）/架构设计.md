本模块由三个独立的 final class 组成，均遵循 ObservableObject + static let shared 的单例模式，作为 SwiftUI 视图层可订阅的全局状态源：
- ThemeManager：通过 UserDefaults 持久化 themeMode，@Published mode 变化时自动写回 defaultsKey "themeMode"。
- ShareExtensionHandler：使用 NSUserDefaults(suiteName: "group.com.voicereader.app") 跨 App Group 读取 pendingURL / pendingText，消费后删除并 synchronize()，避免重复处理。
- ErrorHandler：暴露 @Published currentAlert (AlertInfo)，handle(_:context:) 将 LocalizedError 或 localizedDescription 打印到控制台并以 MainActor 发布 AlertInfo 触发 UI；log(_:level:) 提供带 emoji 前缀的多级别日志。
依赖方向单向：三者仅依赖 Foundation/Combine/SwiftUI，不反向依赖业务层，被上层以 shared 直接调用。
