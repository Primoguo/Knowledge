本模块为纯 SwiftUI 视图层，围绕 `Document` 模型与 `SpeakerViewModel` 组织：
- `ContentView` 作为应用根 TabView 容器，持有 `SpeakerViewModel`、`ErrorHandler`、`ShareExtensionHandler` 单例，并在 `onAppear` / `willEnterForegroundNotification` 中处理来自 Share Extension 的待处理 URL/文本，调用 `TextExtractionService` 提取后通过 `@Environment(\.modelContext)` 写入 SwiftData。
- `DocumentListView` 使用 `@Query(sort: \.lastOpenedDate, order: .reverse)` 驱动文档列表，负责三种导入路径：① 弹出 `DocumentPicker` 选择本地文件；② 弹窗输入 URL 异步抓取网页；③ 接收已选文件 URL 后由 `TextExtractionService.extractText(from:)` 解析文本并插入 `modelContext`。
- `DocumentRowView` 是无状态展示组件，根据 `document.fileType.iconName/iconColor` 渲染图标与标签，并根据 `speakerVM.currentDocument` 与播放状态高亮当前项。
- `UIKit.DocumentPicker` 通过 `UIViewControllerRepresentable` 桥接 `UIDocumentPickerViewController`，在 Coordinator 中使用 `startAccessingSecurityScopedResource()` + `FileManager.copyItem` 将沙盒外文件复制到 `.documentDirectory`，再回调上层。
依赖方向单向向下：Views → UIKit 桥接 / TextExtractionService / SpeakerViewModel / SwiftData，不反向依赖业务逻辑。
